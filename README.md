# Seguidor_De_Actividades_Empresariales
Repositorio con la finalidad de ser un puente de contribuciones entre un servidor Gitea privado y mi portafolio de GitHub, diseñado para reflejar mi flujo de trabajo diario sin exponer código propietario.

Este repositorio utiliza un script personalizado que extrae el volumen de cambios ($files\_changed$, $+insertions$, $-deletions$) de mi instancia local de Gitea y genera un registro de actividad equivalente aquí, manteniendo la integridad del acuerdo de confidencialidad (NDA) con mi empresa.
